# German Crime Analytics Dashboard

An interactive dashboard for analyzing crime statistics across German cities from 2019-2024.

![Dashboard Screenshot](assets/dashboard-screenshot.png)

## Features
- Interactive crime maps
- Crime trends analysis by city
- Crime type distribution
- Seasonal patterns
- Year-over-year comparisons
